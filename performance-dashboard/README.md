# 第二十一事业部业绩看板

静态网页看板，数据来自 `业绩看板汇总表.xlsx`。

## 使用方式

直接部署本目录全部文件到 GitHub Pages 或任意静态网站服务：

- `index.html`
- `styles.css`
- `app.js`
- `data.js`
- `README.md`

所有消耗指标默认使用 `非赠款消耗`。

## 板块

- 仪表盘：总数据、业绩通报、日耗、业务与端口
- 日报：本地推、代充值、代运营数据通报
- 消耗明细：筛选、搜索、排序、导出
- 毛利板块：跳转到既有毛利页面
- 客户构成：直签/渠道数量、消耗、TOP、等级、自运营消耗
- 新增&流失：上传当日数据识别新商机，手工标记并保存到浏览器本地
